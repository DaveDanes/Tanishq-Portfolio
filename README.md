## 🎨 Portfolio Features

- **Modern UI/UX**: Clean dark-themed aesthetic with a vibrant accent gradient.
- **Responsive Design**: Mobile-first approach ensuring a seamless experience across all devices.
- **Scroll Animations**: Smooth staggered reveal animations using `IntersectionObserver` for an engaging user experience.
- **Custom Particles Engine**: A lightweight vanilla JavaScript particles background engine to add subtle interactive depth.
- **Dynamic Typing Effect**: Engaging hero text sequence with responsive backspacing.
- **Zero Dependencies**: Pure HTML5, CSS3, and modern Vanilla JS (No heavy web frameworks used).
